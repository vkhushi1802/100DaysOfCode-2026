## 1. Missing Number
A problem that teaches array traversal and logical reasoning techniques by identifying the missing value from a sequence of numbers.

Given an array containing n distinct numbers from the range: 0 to n, find the number that is missing from the array.

This problem is commonly asked in interviews and helps build concepts like:
- Array processing
- Efficient traversal
- Mathematical reasoning
- Optimization techniques

which are important for solving data-processing problems efficiently.

**Your task:** Find the missing number using an optimized approach.

### Examples

**Input:**
```
Array = [3, 0, 1]
```
**Output:**
```
2
```

---

**Input:**
```
Array = [0, 1]
```
**Output:**
```
2
```

---

**Input:**
```
Array = [9, 6, 4, 2, 3, 5, 7, 0, 1]
```
**Output:**
```
8
```

## 2. Find the Sum of Array Elements
A problem that introduces array traversal techniques and teaches how to efficiently process every element while maintaining a running total of values present in an array.

Given an array of integers, the task is to find the sum of all elements in the array.

This problem appears frequently in interviews and real-world applications like analytics processing, financial calculations, reporting systems, and data aggregation tasks. The challenge involves traversing the array efficiently and accumulating the result.

This teaches:
- Array traversal
- Accumulation logic
- Variable tracking
- Optimization techniques

which are essential for efficient data processing and interview problem solving.

**Your task:** Find the sum of all array elements using an optimized approach.

### Examples

**Input:**
```
Array = [12, 35, 1, 10, 34, 1]
```
**Output:**
```
93
```

---

**Input:**
```
Array = [10, 5, 10]
```
**Output:**
```
25
```
