## 1. Most Common Word (LeetCode #819)
A problem that focuses on processing a paragraph to find the most frequently occurring word that is not present in the banned list. Given a paragraph and a list of banned words, return the most common word that is not banned.
Ignore punctuation and letter casing while processing the text.

This problem helps build concepts like:
- String traversal
- Word extraction
- Character processing
- Basic string manipulation

These concepts are useful for solving text-processing problems.

**Your task:** Find the most common non-banned word using an optimized approach.

### Examples

**Input:**
```
paragraph="Bob hit a ball, the hit BALL flew far after it was hit." , banned=["hit"]
```
**Output:**
```
"ball"
```

---

**Input:**
```
paragraph="a." , banned=[]
```
**Output:**
```
"a"
```

## 2. Better String
Given two strings s1 and s2 of equal length, determine which string contains more distinct subsequences. If both contain the same number of distinct subsequences, return s1.

This problem helps build concepts like:
- String comparison
- Subsequence understanding
- Character traversal
- Basic string manipulation

These concepts are useful for solving a variety of string-based problems.

**Your task:** Determine which string has more distinct subsequences using an optimized approach.

### Examples

**Input:**
```
s1="abc", s2="aac"
```
**Output:**
```
"abc"
```

---

**Input:**
```
s1="abab", s2="abba"
```
**Output:**
```
"abab"
```

---

**Input:**
```
s1="x", s2="y"
```
**Output:**
```
"x"
```
