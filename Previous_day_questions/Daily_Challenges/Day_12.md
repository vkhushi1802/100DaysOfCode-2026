## 1. Number of Lines To Write String (LeetCode #806)
A problem that focuses on calculating how many lines are required to write a string when each character occupies a specified width. Given an array widths representing the width of each lowercase English letter and a string s, determine the total number of lines needed and the width occupied by the last line. Each line can contain at most 100 units.

This problem helps build concepts like:
- String traversal
- Character width calculation
- Basic simulation
- Implementation skills

These concepts are useful for solving implementation-based string problems.

**Your task:** Calculate the number of lines required to write the string using an optimized approach.

### Examples

**Input:**
```
Widths=[10,10,10...] , s = "abcdefghijklmnopqrstuvwxyz"
```
**Output:**
```
[3,60]
```

---

**Input:**
```
Widths=[10,10,10...] , s= "bbbcccdddaaa"
```
**Output:**
```
[2,4]
```

## 2. Smallest Distinct Window
A problem that focuses on finding the shortest substring containing every distinct character present in the given string. Given a string str, return the length of the smallest substring that contains each distinct character of the string at least once.

This problem helps build concepts like:
- String traversal
- Substring processing
- Character comparison
- Basic string manipulation

These concepts are useful for solving a variety of string-based problems.

**Your task:** Find the length of the smallest valid substring using an optimized approach.

### Examples

**Input:**
```
Str= "abcda"
```
**Output:**
```
4
```

---

**Input:**
```
Str= "zzxyzz"
```
**Output:**
```
3
```

---

**Input:**
```
Str= "abcaac"
```
**Output:**
```
3
```
