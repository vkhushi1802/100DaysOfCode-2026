## 1. Remove Element (LeetCode #27)
A problem that strengthens understanding of in-place array manipulation by removing all occurrences of a specified value without using extra space.

Given an integer array and a target value called Val, the objective is to remove all occurrences of Val from the array in-place and return the count of remaining elements. The order of elements that are not removed does not need to be preserved.

This is a classic interview problem that emphasizes pointer-based techniques and in-place modification strategies.

This problem helps build concepts like:
- In-place array modification
- Two-pointer technique
- Index tracking
- Space-optimized traversal

which are fundamental for solving array manipulation problems efficiently.

**Your task:** Remove all occurrences of a given value from the array in-place using an optimized approach and return the number of elements remaining.

### Examples

**Input:**
```
Array = [3, 2, 2, 3], val = 3
```
**Output:**
```
2 (remaining array: [2, 2])
```

---

**Input:**
```
Array = [0, 1, 2, 2, 3, 0, 4, 2], val = 2
```
**Output:**
```
5 (remaining array: [0, 1, 3, 0, 4])
```

---

**Input:**
```
Array = [1], val = 1
```
**Output:**
```
0 (remaining array: [])
```

## 2. Find the Largest Element in an Array
A problem that introduces array traversal techniques and teaches how to efficiently track the largest element while processing all values in an array.

Given an array of integers, the task is to find the maximum element present in the array.

This problem appears frequently in interviews and real-world applications like ranking systems, analytics processing, monitoring platforms, and data evaluation tasks. The challenge involves maintaining the largest value efficiently during a single traversal while handling negative numbers and edge cases.

This teaches:
- Array traversal
- Comparison logic
- Variable tracking
- Optimization techniques

which are essential for efficient data processing and interview problem solving.

**Your task:** Find the largest element in an array using an optimized approach.

### Examples

**Input:**
```
Array = [12, 35, 1, 10, 34, 1]
```
**Output:**
```
35
```

---

**Input:**
```
Array = [10, 5, 10]
```
**Output:**
```
10
```
